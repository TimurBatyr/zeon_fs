_______________________________Algorithm__________________________________
    fs_base_path = .zeon_fs
    ФС = Файловая система

    init_fs: -> None
    Инициализация файловой системы
       Если директория уже найдена
            Ничего не делаем
        Иначе
            Создаем директорию

    checking_database: -> bool
    Проверка целостности БД
        Если в БД нет
            Добавление в БД
        Если в ФС нет
            Перезапись БД

    init_db: -> None
    Инициализация базы данных
        Если база данных найдена:
            Ничего не делаем
        Иначе
            Создать базу данных в директории fs_base_path

    hash_file: -> str
    Хеширование  файла
        Вернет хеш файла

    get_by_key_from_index: -> str
    Получение значения с помощью ключа по индексу
        Если индекс есть в индексе бд
            Если ключ есть в бд[индекс]
                Вернет значение
            Иначе
                Ничего не делает
        Иначе
            Ничего не делает


    add file: -> bool
    Добавление файла в ФС
        Получаем хеш функцию от переданного файла
        Если хеш присутствует в индексе бд то
            Ничего не делаем
        Иначе
            Добавляем в файл
            Добавляем в базу

    delete_file: -> bool
    Удаление файла из ФС
        Если название присутствует в индексе бд
            Удаляем из БД
            Удаляем из ФС
        Иначе
            Ничего не делаем

    get_file: -> bool
    Копирования файла
        Если название присутствует или хеш в бд
            Ничего не делаем
        Иначе
            Копируем файл и переносим в ФС


     list_files: -> list
     Список файлов ФС
         Получить названия из БД1
         Вернем список названий
